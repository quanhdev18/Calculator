# 📱 Calculator App (Kotlin)

Ứng dụng máy tính đơn giản được xây dựng bằng **Kotlin** và **Android Studio** sử dụng `ConstraintLayout`.

## 🚀 Tính năng

- Cộng, trừ, nhân, chia
- Giao diện tối giản, dễ sử dụng
- Xử lý biểu thức theo thứ tự ưu tiên
- Tự động cập nhật kết quả theo từng bước nhập

## 🛠️ Công nghệ sử dụng

- Ngôn ngữ: **Kotlin**
- Android SDK: **>= 24**
- UI: `ConstraintLayout`, `ViewBinding`
- IDE: **Android Studio Hedgehog / Iguana**
